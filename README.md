# Plugin WeaponsAttributes - MC 1.20.1

Fix weapons attributes :
- fix lore
- fix attack damage and attack speed on weapons with attributes

## Commands

- **/fix-attributes**  
Fix items attributes and update lore.

## Permissions

**Nothing.**
